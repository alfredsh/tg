# 🔐 TG Secret Chat — Android APK

Telegram-клиент с поддержкой секретных чатов. Собирается автоматически через GitHub Actions.

---

## 🚀 Как получить APK — пошагово

### Шаг 1 — Создать репозиторий на GitHub

1. Зайдите на **github.com** → кнопка **"New repository"**
2. Назовите его, например: `tg-secret-chat`
3. Выберите **Private** (чтобы ваши ключи не светились)
4. Нажмите **"Create repository"**

---

### Шаг 2 — Загрузить файлы

**Вариант А — через сайт GitHub (проще):**
1. В репозитории нажмите **"uploading an existing file"**
2. Перетащите все файлы из этой папки:
   - `main.py`
   - `buildozer.spec`
   - `.github/workflows/build.yml`
3. Нажмите **"Commit changes"**

**Вариант Б — через терминал:**
```bash
git init
git remote add origin https://github.com/ВАШ_НИК/tg-secret-chat.git
git add .
git commit -m "Initial commit"
git push -u origin main
```

---

### Шаг 3 — Дождаться сборки

1. После пуша автоматически запустится сборка
2. Перейдите в репозиторий → вкладка **"Actions"**
3. Вы увидите запущенный воркфлоу **"Build Android APK"**
4. ⏳ Первая сборка занимает **25–40 минут** (скачивает Android SDK)
5. Повторные сборки — **5–10 минут** (благодаря кешу)

---

### Шаг 4 — Скачать APK

**После успешной сборки:**

- Вкладка **Actions** → ваша сборка → раздел **"Artifacts"** → скачать `TGSecretChat-debug.apk`

**ИЛИ** (если сборка из `main`):

- Вкладка **Releases** → последний релиз → скачать `.apk` файл

---

## 📱 Как использовать приложение

1. Установите APK на Android (разрешите установку из неизвестных источников)
2. Получите `api_id` и `api_hash` на **my.telegram.org → API Development Tools**
3. Введите их на первом экране приложения
4. Войдите через номер телефона + SMS-код
5. Найдите пользователя через `@username`
6. Нажмите **"Создать секретный чат"**
7. Попросите собеседника **принять запрос** в официальном Telegram

## ⚠️ Важные ограничения секретных чатов

- Принять запрос можно **только** в официальном Telegram (iOS/Android/Desktop)
- Секретный чат **не синхронизируется** между устройствами
- APK подписан **debug-ключом** — для публикации в Google Play нужен release-ключ
